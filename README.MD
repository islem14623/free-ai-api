# Free AI Assistant API

AI-powered text processing API using Google Gemini (100% FREE!)

## Features

- ✅ Text Summarization
- ✅ Multi-language Translation (French, English, Arabic, 100+ languages)
- ✅ Auto-detect source language
- ✅ Powered by Google Gemini AI
- ✅ Auto-generated API documentation

## Tech Stack

- FastAPI
- Google Generative AI (Gemini)
- Python 3.10+

## API Endpoints

- `GET /` - Welcome message
- `GET /health` - Health check
- `GET /test` - Test AI connectivity
- `POST /summarize` - Summarize text
- `POST /translate` - Translate between languages
